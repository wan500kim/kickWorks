## **kickWorks 비속어 순화 필터**

해당 프로젝트는 딥러닝을 통한 비속어 필터링 서비스를 제공합니다.

웹에서 구동되며 비속어로 판단된 어절을 같은 글자 수의 **순화된 단어로 치환**합니다.
